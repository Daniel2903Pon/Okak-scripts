# Okak-scripts
Приватный скрипты от t.me/okak_pon_okak(t.me/okak_scripts)

# Список скриптов
CMHelper - Хелпер для центрального рынка. Функционал: ловля лавок(при слете/свободной), рендер на свободные лавки, радиус лавок. Цена - 300 руб/месяц, 900 руб/навсегда.
fermabot - Бот на начальную ферму. Подходит для прохождения начальных квестов либо работ в деморгане. Функции: автоходьба, автокапча. Бот портирован с ПК скрипта. Цена: 200руб/месяц 800руб/навсегда.
minebot - Бот на шахту. Возможности: телепорт/ходьба к рудам, рендер на руды, проверка есть ли уже другой игрок около руды. Цена скрипта - 300руб/месяц 800руб - навсегда
eclipsemain - Порт бота на шахту
videocard lovlya - Скрипт автоматически покупает за вас видеокарту во время слета. Настраиваемая задержка, клики и дебаг мод. Скрипт сам решает за вас капчу. Цена скрипта - 300руб/месяц
